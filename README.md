> **Risk of Rain 2의 성장 루프 × Space Marine 2의 전투 감각 — 좀비 버전**

좀비 대군을 처형 연쇄로 쓸어담으며 카드를 쌓아 강해지고, 5웨이브마다 보스를 처치하는 **TPS 웨이브 생존 로그라이트**.

---

## 게임 컨셉

| 항목 | 내용 |
|---|---|
| **장르** | TPS 웨이브 생존 + 로그라이트 업그레이드 |
| **엔진** | Unreal Engine 5.4 |
| **언어** | C++ / Blueprint |
| **플랫폼** | PC |
| **개발 기간** | 2026-05-01 ~ 2026-05-27 (4주) |
| **팀 구성** | 5인 (전원 프로그래머) |

### 레퍼런스
- **Risk of Rain 2** — 웨이브 클리어 후 아이템 선택, 스택 성장 빌딩
- **Space Marine 2** — 좀비 대군 압도, 처형 연쇄, 묵직한 타격감

### 게임 필러

| 필러 | 설명 |
|---|---|
| **압도** | 수백 마리 좀비 대군에 둘러싸이는 스케일감 |
| **성장** | 웨이브마다 강해지는 카드 업그레이드 빌딩 |
| **타격감** | 근접·원거리 전투의 묵직한 피드백 |

### 시그니처 메카닉
- **처형 연쇄** — 체력 임계값 이하 좀비 처형 시 주변 경직 → 연속 처형
- **과열 시스템** — 원거리 무기 과열 → 근접 강제 전환 → 냉각 후 복귀
- **카드 업그레이드** — 웨이브 클리어 후 3장 중 1장 선택 (화력 / 생존 / 이동)

---

## MVP 범위

### 구현 예정
- 플레이어 TPS 이동 + 원거리 사격
- 근접 전투 (기본 콤보 + 처형 연쇄)
- 좀비 웨이브 5라운드 (일반 / 돌진형 / 탱커형)
- 보스 1마리 (5라운드)
- 웨이브 클리어 후 업그레이드 카드 선택
- 무기 과열 시스템
- 아레나 맵 1개 (K-좀비 테마 — 도시 폐허 + 지하철)
- 메인 메뉴 + HUD (체력 / 탄약 / 과열 게이지 / 웨이브 카운트)

### 미구현 (스코프 제외)
- 멀티플레이
- 세이브 시스템
- 랜덤 맵 생성
- 스토리 / 컷씬
